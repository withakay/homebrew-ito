# Homebrew Tap for Ito

This is the official [Homebrew](https://brew.sh) tap for [Ito](https://github.com/withakay/ito).

## Installation

```bash
brew tap withakay/ito
brew install ito
```

Or install directly:

```bash
brew install withakay/ito/ito
```

## Updating

```bash
brew update
brew upgrade ito
```

## Building from HEAD

To install the latest development version:

```bash
brew install --HEAD withakay/ito/ito
```

This requires Rust to be installed.
